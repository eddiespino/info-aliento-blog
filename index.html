Deployment Instructions

The automatic build process encountered some issues. To deploy manually:

  1. Clone your repository to your local machine
  2. Run npm run build to create a production build
  3. Copy the contents of dist/public to a new docs folder
  4. Push the docs folder to GitHub
  5. Enable GitHub Pages in repository settings

About This Application

This application allows you to view Hive blockchain witnesses and vote for them using Keychain.
